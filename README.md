# Install

## Clone

```
git clone https://github.com/LeonardoFerreiraa/dotfiles.git ~/.dotfiles
```

## Fonts

```
ln -s ~/.dotfiles/fonts ~/.fonts
```

## I3

```
ln -s ~/.dotfiles/i3 ~/.config
```

## VIM

```
ln -s ~/.dotfiles/vimrc ~/.vimrc
```
